Solution :

Time Complexity :


```java
int findOnce(int arr[], int n)
{
    int i,xor=0;
    for(i=0;i<n;i++)
        xor=xor^arr[i];
    return xor;
}
```


```java
int findOnce(int arr[], int n)
{
    int i;
    if(n==1)
        return arr[n-1];
    for(i=0;i<n-1;i+=2)
    {
        if(arr[i]!=arr[i+1])
        {
            return arr[i];
        }
    }
    return arr[n-1];
}
```
